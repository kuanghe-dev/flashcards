## Project Overview

`flash-cards` is an early-stage project for learning Chinese. It reads Chinese characters from text files in the `assets` subdirectory and displays them in flash card style.

## Repository Structure

- `assets/` — Text data files containing Chinese characters, named by grade level (e.g., `grade1b.txt`).

## Data File Format

Asset files in `assets/` contain Chinese characters encoded in UTF-8:

```
1→这次他们很快
2→
```
